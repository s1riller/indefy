Вопрос 1: Что такое инженерно-авиационное обеспечение и какие задачи оно решает?

1. Инженерно-авиационное обеспечение - это комплекс мероприятий, направленных на поддержание авиационной техники в исправности и готовности к боевым действиям.
2. Инженерно-авиационное обеспечение обеспечивает высокую эффективность применения авиационной техники, ее боевую готовность и подготовку.
3. Инженерно-авиационное обеспечение осуществляется инженерно-авиационной службой авиации.
4. Задачи инженерно-авиационного обеспечения включают поддержание технической готовности авиационной техники, увеличение ее эффективности, выполнение специальных задач и боевой подготовки.

Вопрос 2: Какие требования ФАП ИАО являются обязательными для личного состава авиационных объединений и организаций, имеющих государственную авиацию?

def calculateAngle(entityPositionX, entityPositionY, entityPositionZ, localPlayer):

localPlayerPositionX = pm.read_float(localPlayer + 0x138)
localPlayerPositionY = pm.read_float(localPlayer + 0x138+4)
localPlayerPositionZ = pm.read_float(localPlayer + 0x138+8)

deltaX = entityPositionX - localPlayerPositionX
deltaY = entityPositionY - localPlayerPositionY
deltaZ = entityPositionZ - localPlayerPositionZ

ah = math.sqrt(deltaX*deltaX + deltaY*deltaY)

yaw = math.atan2(-deltaZ, ah) * 180.0 / math.pi
pitch = math.atan2(deltaY, deltaX) * 180.0 / math.pi

NormalizeViewAngles(yaw, pitch, 0)

return NormalizeViewAngles(yaw, pitch, 0)

def NormalizeViewAngles(x, y, z):
if (x > 89):
x = 89
if (x < -89):
x = -89

while (y > 180):
y -= 360
while (y < -180):
y += 360

if (z != 0):
z = 0

return x, y, z
